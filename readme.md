# Akatsuki – AI Moderated Community Chat Platform

## Features
- Real-time messaging using WebSockets
- AI-powered NSFW and hate speech detection
- User authentication and moderation system
- Multi-channel communication

## Tech Stack
- Python, Flask
- WebSockets
- SQLite / PostgreSQL
- Hugging Face API
- HTML, CSS, JavaScript

## Purpose
Akatsuki is a safe community platform designed with automated AI moderation to prevent harmful content and ensure secure user interactions.
